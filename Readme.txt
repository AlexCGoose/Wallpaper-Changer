Setup steps:
1. Add wallpaers to the Wallpaers folder, numbered sequentially (i.e. 100 wallpapers numbered 1 to 100) - 5 wallpapers have been included to demonstrate
2. Add index values of sunrise and sunset wallpaper to Wallpaper Index.txt
3. Add your current latitude and longitude to Location.txt
4. Create a shortcut of wallpaper-changer.pyw and move it to %APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup
