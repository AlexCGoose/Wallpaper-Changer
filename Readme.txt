Setup steps:
1. Add wallpapers to the Wallpapers folder, numbered sequentially (i.e. 100 wallpapers numbered 1 to 100) - 5 wallpapers have been included to demonstrate
    - DO NOT ADD ANY OTHER FILES TO THIS FOLDER
2. Add your latitude, longitude, and the index values of the desired sunrise and sunset wallpaper to config.json
4. Create a shortcut of wallpaper-changer.pyw and move it to %APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup
5. Run the file from the Startup folder or restart your computer
